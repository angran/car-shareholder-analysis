## 核心结论（摘要）

1. **结构性极度集中**：SRS Investment Management 单一持股 ~49%，其总裁 Jagdeep Pahwa 即为 Avis 执行董事长 —— 这是公开、备案的"一致行动人"关系，有正式的 *Cooperation Agreement*（2025-09 修订，投票上限从 35% 提高到 45%）。
2. **浮筹极少**：总股本仅约 3,526 万股，扣除 SRS 锁仓部分后，实际可流通股非常有限。
3. **空头超过 100% of float**：空头利息历史性地超过可借股数，为轧空（short squeeze）提供了完美燃料。
4. **Pentwater Capital** 2026-03-06 备案 13G/A，持股 12.3%（含期权 25.47 万股），申报为**被动投资人**，未披露与 SRS 的一致行动关系。
5. **"联合做局"判断**：没有任何公开文件显示 SRS 与 Pentwater 存在协同。但结构上——大股东深度绑定管理层、float 被锁定、空头被逼仓——客观上形成了一个"天然做局"的场景，无需违法合谋即可推动股价暴涨。真正的法律风险点在于：任何机构若在未披露 group 关系的情况下实际协调买入，都可能触发 13D 补报和市场操纵调查。
